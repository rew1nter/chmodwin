# why
`chmod 600`, for example, .pem file in windows. Then use it to ssh just like linux from your windows terminal.

# help
$ chmod 
```
This can 'chmod 600' files in Windows
usage:
        chmodwin key.pem
        chmodwin.exe key.pem
```
